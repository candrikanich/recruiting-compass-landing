# Icons Component Directory

This directory contains reusable SVG icon components for the landing site.

## Usage

```vue
<template>
  <CompassIcon class="w-6 h-6 text-primary-600" />
</template>

<script setup>
import CompassIcon from '~/components/icons/CompassIcon.vue'
</script>
```

## Available Icons

- `CompassIcon.vue` - Main logo/compass icon placeholder
- Add more icons as needed for the landing site

## Design System

All icons should use consistent sizing and color classes that work with the Tailwind CSS design system.
